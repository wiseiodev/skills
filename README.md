# skills

Repo to store + share agent skills.

## Install

Install a skill with:

```bash
npx skills add wiseiodev/skills/<skill>
```

## Available skills

### graphite

```bash
npx skills add wiseiodev/skills/graphite
```

### gt

npx skills add wiseiodev/skills/gt
